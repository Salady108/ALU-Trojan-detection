# Generic ALU with Hardware Trojan Study

This repository contains a **Verilog-based generic ALU** developed during a **24-hour team hackathon**.  
The project focuses on **RTL design and functional verification**, along with a **conceptual team-based study** of the impact of hardware Trojans on circuit behavior.
---
## Overview
- Designed a **parameterized ALU** using Verilog HDL
- Verified functionality through simulation and waveform analysis
- Studied behavioral differences between clean and Trojan-inserted ALU variants
---
## My Contribution
- Implemented the **RTL design of the ALU** in Verilog
- Performed functional verification using simulation and waveform inspection
- Gained conceptual exposure to Trojan impact analysis through team collaboration
---
## Tools
- **HDL:** Verilog  
- **Simulation & Synthesis:** VS Code  
- **Waveform Viewing:** GTKWave  
- **Analysis (team-based):** Python (Matplotlib)
